## Before you start
python[3][.exe] -m pip install --upgrade pip

### Requirements
pip install 
numpy
scipy
astropy
matplotlib==3.5.0
configparser
ipython
pyqt5

### Start QtYETI
• Navigate to the directory where QtYETI has been extracted to  
• Use "python ./QtYeti.py"
